2023-03-22

ffmpeg, 剪辑

ffmpeg参数格式

剪辑视频

-i : 输入文件
-ss : 起始时间
-t : 指定时间长度，默认单位为秒
-to : 指定结束时间
-c:v : 视频编码

-c:v copy保持原本编码不变
-c:v h264_nvenc可以使用nvdia硬件编码

视频转码

如果编码方式不支持硬件编码，则修改-c:v参数
-b:v : 视频码率，一般6000足够清晰
-c:a : 音频编码
-b:a : 音频码率
